À faire vous-même 7.1

Lisez ou Relisez le document consacré au réseau (à consulter ici)

Créer un questionnaire (sous forme de page web) destiné à vérifier que les principales notions développées dans ce document ont bien été comprises et acquises.

Vous êtes totalement libre :

  • pour la mise en page de ce questionnaire
  • pour le choix des questions
  • pour le type de questions (QCM, question ouverte,...)
  • pour le nombre de questions

Seule contrainte : une fois que l'utilisateur aura répondu à toutes les questions, une note (sur 20) devra lui être accordée. Un commentaire lié à la note pourra aussi être donné ("bon travail", "insuffisant, vous devez relire attentivement le document",...)

  • la documentation officielle de jQuery est très bien faite : ici
